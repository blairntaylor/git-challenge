How to Git Add, Commit, Push

  1. Go to Github.com
  2. Go to profile dropdown menu
  3. Go to your repositories and click on 'green" New
  4. Create new repository name ex. HARD-GIT-SH*T and go to OK
  5. Go to 2nd section under "... create a new repo" and copy section of code
  6. Open GitBash on local computer
  7. Navigate to desktop > cd desktop
  8. Make a new directory > mkdir challenges
  9. Navigate into > cd challenges
  10. Open to see it is empty > ls
  11. Paste (line 5 of the directions) comments from Github into GitBash and enter
  12. Use command ls to see what is inside ex. README.md
  13. Create a new file > touch index.html
